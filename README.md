# Practical_App3: Bank Marketing Classifier Comparison
Comparing Classifiers using dataset of "Bank Marketing" in UCI Machine Learning repository

## Overview
Comparison of machine learning classifiers to predict term deposit subscriptions.

## Key Findings
- Random Forest performed best with 91.2% accuracy and 0.58 F1 score
- Campaign timing (March, September-December) significantly impacts conversion
- Students and retired individuals have highest conversion rates

## Recommendations
- Focus campaigns on high-conversion months
- Target student and retired segments
- Implement model for lead scoring

## Files
- `bank_marketing_classifier_analysis.ipynb`: Complete analysis
- `best_bank_marketing_model.pkl`: Trained model
- `preprocessor.pkl`: Data preprocessing pipeline
